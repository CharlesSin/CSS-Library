## Charles CSS Library
### Clone File
```
git clone https://github.com/CharlesSin/CSS-Library.git
```

### Go to directory
```
cd CSS-Library
```

### Install node package
```
npm install
```

### Run gulp command to complie SCSS file
```
gulp
```
